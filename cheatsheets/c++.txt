Data types:

Fundamental types in C++ are divided into three categories: integral, floating point, and void. Integral types are capable of handling whole numbers. Floating point types are capable of specifying values that may have fractional parts. The void type describes an empty set of values.
The category integral includes the following datatypes: char, bool, short, int, long, long long, wchar_t, __wchar_t
The category floating point includes the following datatypes: float, double, long double

constexpr:
The constexpr specifier declares that it is possible to evaluate the value of the function or variable at compile time. 

Notes:

- constexpr specifier is required when initializing static data members of non-integral types
